TCG Collector is a simple web app to log and view one's trading card collection.
Currently TCG Collector is built only for Yu-Gi-Oh! but is planned to be expanded to other card games.


How to use:

Search for the title of the card you wish to add to your collection, and click the card from the results page.
There is a 'collection' column under the sets table where you can edit the number of cards you have of each set number.
Cards can be sorted into decks for ease of viewing.
